Quick Start Guide

1. Open a requirements spec above with your feature name.
2. PM/Designer fills in flows, screenshots, and requirements per screen.
3. Engineer adds API calls, edge cases, error handling.
4. Click "Copy prompt for Claude Code" and paste into Claude Code with the /feature-scaffold skill.
5. Review the proposed architecture in the tree preview editor.
6. Claude Code generates all production code.
